# Prompt de Proyecto Web Multilenguaje con Integración de Mercado Pago y SEO Optimizado

## Descripción general  
El objetivo es desarrollar un sitio web moderno, funcional al 100%, optimizado para SEO, con integración completa de carrito de compras y pagos a través de Mercado Pago. El público objetivo son familias de poder adquisitivo medio/alto, buscando experiencias de usuario y diseño de alta calidad, siguiendo buenas prácticas de desarrollo, accesibilidad y usabilidad.

## Alcance y requerimientos

1. **Edición de Contenido**  
   - Todo el contenido debe ser editable directamente desde archivos (ej: Markdown, JSON, YAML, etc.), facilitando la gestión y actualización del sitio sin depender de un CMS externo.

2. **E-commerce y Mercado Pago**  
   - Carrito de compras completo, funcional y seguro.
   - Integración nativa con Mercado Pago para procesar pagos.
   - Flujo de compra claro, intuitivo y confiable (agregar/quitar productos, checkout, pago, confirmación).
   - Soporte para promociones y cupones (ideal).

3. **Multilenguaje**  
   - El sitio debe estar disponible en español e inglés.
   - La estructura debe permitir fácilmente agregar más idiomas en el futuro.
   - Todo el contenido, navegación, mensajes y metadatos deben estar traducidos.

4. **Modos de visualización**  
   - Implementar modo claro y modo oscuro, permitiendo al usuario alternar entre ambos de forma sencilla.

5. **Buenas prácticas de desarrollo**  
   - Código modular, limpio y documentado.
   - Uso de componentes reutilizables.
   - Responsivo, accesible (WCAG nivel AA como mínimo).
   - Uso de frameworks y librerías modernas (React, Next.js/Vue/Nuxt, o equivalente).

6. **Optimización SEO**  
   - Metadatos completos y dinámicos por página (title, description, OpenGraph, etc.).
   - URLs legibles y bien estructuradas.
   - Microdatos/JSON-LD para productos y servicios.
   - Sitemap.xml y robots.txt configurados.
   - Carga rápida (Core Web Vitals).
   - Imágenes optimizadas.
   - Accesibilidad y experiencia de usuario que potencien el posicionamiento.
   - Compatible con Google Analytics, Search Console, y herramientas de seguimiento.

## Inspiración y Benchmarks

- [terappio.com/ar](https://www.terappio.com/ar): modelo de servicios y estructura.
- [cursos.soymamaypediatra.com](https://cursos.soymamaypediatra.com/): público objetivo y experiencia de usuario.
- [pediatramel.com.ar](https://pediatramel.com.ar/): tono y calidad de contenido.

## Entregables

- Estructura completa del sitio, funcional, lista para producción (no MVP, no versión beta).
- Código fuente íntegro, siguiendo buenas prácticas.
- Documentación técnica y de uso para mantenimiento y escalado.
- Instrucciones claras para localización y edición de contenido.
- Integración totalmente operativa con Mercado Pago.
- Sitio publicado y testeado en ambos idiomas, con ambos modos de visualización.

## Deadline

- **Primer release funcional**: 31 de octubre de 2025.
